Welcome our Physically-based Melting Simulation

This is the final project for UPenn's Physically-based Simulation class with Joe Kider.

References

We are basing our implementation off the following papers:

  • R. Bridson, M. Fischer, “Fluid Simulation SIGGRAPH 2007 Course Notes,” SIGGRAPH Course (SIGGRAPH 2007).

  • K. Iwasaki, H. Uchida, Y.Dobashi, T. Nishita, "Fast Particle-based Visual Simulation of Ice Melting," Computer Graphics Forum (PG 2010).

  • A. Paiva, F Petronetto, T. Lewiner, G. Tavares, “Particle-based non-Newtonian ﬂuid animation for melting objects,” XIX Brazilian Symposium on Computer Graphics and Image Processing (SIBGRAPI 2006).

  • D. Terzopoulos, J. Platt, K. Fleischer, “Heating and melting deformable models,” The Journal of Visualization and Computer Animation (1991).

Base code

Our project is built on top of Rama Hoetzlein's Fluid V2 code.
